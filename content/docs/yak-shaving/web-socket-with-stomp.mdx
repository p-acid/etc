## Web Socket

- 브라우저와 서버 간 인터랙티브 통신 세션을 설정할 수 있는 기술
- 개발자는 웹 소켓 API를 사용하여 서버로 **메시지** 전송 가능
- 서버 응답을 받기 위해 폴링을 반복할 필요 없이 이벤트 기반으로 처리 가능
- **실시간 양방향 통신**이 가능하여 채팅, 알림, 실시간 데이터 스트리밍 등에 적합

## STOMP

> **STOMP** is a simple interoperable protocol designed for asynchronous message passing between clients via mediating servers. It defines a text based wire-format for messages passed between these clients and servers.

- 중재 서버를 통해 클라이언트 간 비동기 메시지 전달을 위해 설계된 간단한 상호 운용 프로토콜
- 클라이언트와 서버 간에 전달되는 메시지에 대한 텍스트 기반 와이어 형식을 정의
